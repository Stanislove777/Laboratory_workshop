## Описание проекта

Лабораторный практикум из Университеского курса по программированию.

Каждая лабораторная работа, помимо основного задания, разделена на три части по уровню сложности - `A, B, C`.

**Цель проекта**: Познакомится с синтаксисом и некоторыми библиотеками разных языков.

## Запуск проекта

Для запуска проекта на языке `C` необходимо выполнить, команду:
```sh
cd LabWork_1/C
gcc *.c
```

## Задания
### LabWork_1

**Основное задание**: Для функции `f(x) = x * e^x + 2sinx - sqrt(abs(x^3 - x^2))` составить программу построения таблицы значений функции при изменении аргумента от A = -5 до B = 5 с шагом H = 1,0 . Значения A, B и H объявить как константы в директивах `#define`. В каждой строке выводить значение аргумента и соответствующее ему значение функции в форматированном виде с 4 знаками после запятой.

**Задание A**: Среднее арифметическое таких значений функции, которые меньше числа М.

**Задание B**: Среднее арифметическое таких значений функции, которые имеют нечетную целую часть и дробную часть, большую 0,5.

**Задание C**: Для решения следующих заданий разрешается использовать цикл только один раз. Минимальное среди неотрицательных значений функции и количество таких минимальных значений.

### LabWork_2

**Основное задание**: Для функции из *LabWork_1* составить программу построения таблицы значений функции, состоящую из *n* строк при изменении аргумента от A до B.
В каждой строке таблицы выводить значение аргумента x и соответствующее ему значение функции y в форматированном виде с 4 знаками после запятой. В конце таблицы вывести отдельной строкой среднее арифметическое всех значений функции.

Кроме того, программа должна определить, сколько точек с координатами (x, y) из полученной таблицы попадает внутрь области, ограниченной фигурой, соответствующей индивидуальному заданию.

**Задание A**: В качестве области выступает прямоугольник со следующими вершинами.

(-10, -6), (-10, 4), (0, 4), (0, -6).

**Задание B**: Две области: 

квадрат - (-1, 1), (-1, 0), (0, 0), (1, 0) и четверть окружности - (0, 0), (0, -1), (1, 0).

**Задание C**: Сгенерировать четыре случайные точки плоскости (x1, y1), (x2, y2), (x3, y3), (x4, y4) с тем условием, что точка (x1, y1) принадлежит первой четверти координатной плоскости, (x2, y2) – второй четверти, (x3, y3) – третей четверти, (x4, y4) – четвертой четверти. После этого найти количество точек (x, y) из таблицы, принадлежащих полученной области. Сделать это двумя способами – через уравнения прямых и через площади треугольников.
